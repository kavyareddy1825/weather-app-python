#  Weather App (Python)

A simple weather application that fetches real-time weather data using OpenWeatherMap API.

## Features
- Get live weather details
- Shows temperature, humidity, condition
- User input based city search

##  Tech Stack
- Python
- Requests library
- OpenWeatherMap API

##  How to Run
1. Install dependencies:
   pip3 install requests
2. Add your API key
3. Run:
   python3 weather.py
